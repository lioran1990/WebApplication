About the project:

In this project we created a Blog website, which include all the core features of a general blog. There are posts, comments, and users in the blog. Post can include videos and images built-in. Each user can subscribe to the website and start viewing posts, and use the Search bar as necessary. There is a users and role management engine- each role will decide what should exposed to user.

Teachnical information:

Server side technology:

  • ASP.NET (Freamework 4.5.2) based on Microsoft MVC 5 architecture.
  • The data handled by Entity Framework, and stored on SQL Server.
  • CRUD functionality: Create, Read, Update, Delete of entities.

Client side technology:

  • HTML5: Video, Canvas, and the following page design: Aside, footer, header, nav bar, section.
  • CSS3: Text-shadow, Transition, Multiple-columns, Font-face, Border-radius.
  • More: JQuery,JavaScript, Ajax.
